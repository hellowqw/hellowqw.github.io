MarkDown Editing 简介

Markdown plugin for Sublime Text. Provides a decent Markdown color scheme (light and dark) with more robust syntax highlighting and useful Markdown editing features for Sublime Text. 3 flavors are supported: Standard Markdown, GitHub flavored Markdown, MultiMarkdown.
—— github-MarkDownEditing

支持语法高亮；支持Github Favored Markdown语法；自带3个主题。

Sublime Text 3 安装

  1. Ctrl + Shift + P -> Install Package。
  2. 键入Markdown editing 安装。
  3. 安装完后重启Sublime即可。

更换主题

performance -> package setting -> MarkDown Editing -> change color scheme

更多配置可以移步github-MarkDownEditing
