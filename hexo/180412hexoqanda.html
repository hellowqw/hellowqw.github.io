hexo更改站点配置文件- title后网站标题无变化

网站修改标题后本地预览无变化，经过一搜索，网上的方法有清楚缓存在生成静态页面：

$ hexo clean
$ hexo g

试过之后没反应，一顿折腾之后发现重启服务后可以了。
停止服务

Ctrl + C

启动服务

$ hexo s

如果还是不可以，试试清除缓存。

如何还是不行，尝试重新安装hexo：

  • 保存文章及当前配置（网站配置、主题配置）
