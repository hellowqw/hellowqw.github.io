将hexo主题next改成自己喜欢的样子

发表于 2017-08-27 | 更新于 2018-04-12 | 分类于 hexo |
字数统计 367 | 阅读时长 1

主要内容：修改主题css样式、首页文章显示阅读全文、增加目录

修改主题css样式

主题文件夹下source -> css -> _custom -> custom.styl
可以加入自己想修改的css样式
还可以参考内建标签的使用。

首页文章显示阅读全文

next主题首页默认显示的是文章全部内容，大多数情况下这并不是我们希望的展现形式，可以通过如下方法设置阅读全文而只显示文章摘要。

  1. 在文章中使用 <!-- more --> 手动进行截断，Hexo 提供的方式
  2. 在文章的 front-matter (无法查看可以试试这个front-matter)中添加 description，并提供文章摘录
  3. 自动形成摘要，在 主题配置文件 中添加：
    auto_excerpt:
    enable: true
    length: 150

默认截取的长度为 150 字符，可以根据需要自行设定

建议使用 <!-- more -->（即第一种方式），除了可以精确控制需要显示的摘录内容以外， 这种方式也可以让 Hexo 中的插件更好的识别。

改变目录名字

在主题配置文件_config.yml中修改：

# When running the site in a subdirectory (e.g. domain.tld/blog), remove the leading slash (/archives -> archives)
menu:
    home: /        
    关于我: /about/ #默认为about

更多修改可以在主题目录下languages文件夹中找到当前使用的语言更改。
