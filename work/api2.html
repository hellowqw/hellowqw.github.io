字数统计 712 | 阅读时长 3

注意：接口统一加platform 6.广告主 7媒体主

广告平台

1. 获取二维码接口

请求URL:/Main/action/Login_reg/Login_wechat/login_code

请求方式：POST

请求参数：

名称 类型 是否必填 说明

返回参数：

名称 类型 是否必填 说明
code INT YES 返回状态
msg STRING YES 状态说明
url STRING YES 二维码路径
scend_id STRING YES 场景值

2. 扫码回调接口

请求URL:/Main/action/Login_reg/Login_wechat/code_callback

请求方式：POST

请求参数：

名称 类型 是否必填 说明
scend_id string YES 场景值

返回参数：

名称 类型 是否必填 说明
code INT YES 返回状态
type int YES 状态 0扫码中 1扫码成功未绑定 2登录成功
msg STRING no 状态说明
data array yes 返回数据

data数组：

名称 类型 是否必填 说明
token string YES token
