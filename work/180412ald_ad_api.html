所有接口返回格式统一为JSON

1. 广告主列表接口

请求URL:/Main/action/User/User_public/user_list

请求方式：POST

请求参数：

名称 类型 是否必填 说明
token string YES token
total INT YES 分页条数默认20条
currentPage INT YES 页数

返回参数：

名称 类型 是否必填 说明
code INT YES 返回状态码200成功202失败
msg STRING YES 状态说明
data array YES 数组

data数据：

名称 类型 是否必填 说明
id int YES 唯一id

appraisal Int YES 认证0未认证1审核中 2已认证
appraisal_type Int YES 认证类型1企业2个人
