接口

发表于 2018-04-03 |

备注：

所有接口返回格式统一为JSON

201 登录状态失效

202 返回无数据 接口传入数据错误

200 成功返回

访问本地连接：http://192.168.1.163:8080/ad_project/index.php

1. 广告主列表接口

请求URL:/Main/action/User/User_public/user_list

请求方式：POST

请求参数：

名称 类型 是否必填 说明
token string YES token
total INT YES 分页条数默认20条
currentPage INT YES 页数

返回参数：

名称 类型 是否必填 说明
code INT YES 返回状态码200成功202失败
msg STRING YES 状态说明
data array YES 数组
total int YES 总条数

data数据：
|名称|类型|是否必填|说明|
|:—|:—|:—|:—|
|id|int|YES|唯一id|
|phone|String|YES|账户|
|payee|STRING|YES|企业个人账户名称|
|balance|STRING|YES|现金余额|
|virtual_balance|STRING|YES|虚拟余额|
|money|STRING|YES|账号余额|
|appraisal|Int|YES|认证0未认证1审核中 2已认证|
|appraisal_type|Int|YES|认证类型1企业2个人|
